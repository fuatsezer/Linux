# Dosya İçeriğini Okuma
`cat`: dosyayı okur.
```console
[fuatsezer@localhost notes]$ cat words.txt
```
`less`: dosyayı okur ve dosyanın içerisinde haraket etmemizi sağlar.
```console
[fuatsezer@localhost notes]$ less words.txt
```
`more`: dosyayı okur ve dosyanın içerisinde haraket etmemizi sağlar.
```console
[fuatsezer@localhost notes]$ more words.txt
```
`head -n 5`: dosyanın ilk 5 satırını okur
```console
[fuatsezer@localhost notes]$ head -n 5 words.txt
```
`tail -n 5`: dosyanın son 5 satırını okur
```console
[fuatsezer@localhost notes]$ tail -n 5 words.txt
```
